◆ alteraAssento()

t_assento* alteraAssento ( int  indice,
int  id,
int  valor 
)

Descrição : Função geral para a modificação (alocação/liberação) dos assentos onde são implementados os mecanismos de exclusão mutua.

Entrada:

  • indice: valor que corresponde a posição de um assento especifico no vetor de assentos global.
  • id: identificador do agente executando a função.
  • valor: valor para o qual o atributo ocupante do assento será alterado (zero para liberar ou diferente de zero para alocar)

Saida: Caso o assento já esteja ocupado, retorna NULL. Caso contrario, retorna o assento alocado.

Descrição: recebe um vetor de inteiros representando o estado atual dos assentos como parâmetro e retorna um vetor de inteiros, sendo o seu primeiro elemento o número de assentos livres contados e o restante contendo os índices dos assentos livres dentro do vetor de assentos.
